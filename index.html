Network Eye (NetEye): A web tool for the analysis of complex networks

A preliminary demo that proves that importing a file as a graph and reading it works. With this tool you can import networks and analyze them in order to make useful observations, conclusions or decisions for further study.

You may also drop any file from the local file system or a remote URL anywhere on this page.

Or try one of the predefined networks below

Les Misérables
Coappearance network of characters in the novel Les Misérables, compiled by Donald Knuth.
Zachary's karate club
Zachary's karate club
Social network of a university karate club, described in the paper An Information Flow Model for Conflict and Fission in Small Groups by Wayne W. Zachary. Node 1 is the instructor and node 34 is the club administrator.
